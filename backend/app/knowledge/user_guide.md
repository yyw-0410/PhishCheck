# PhishCheck User Guide

## Email Analysis
1. Export .eml from your email client (Gmail: ⋮ → Download message)
2. Upload to PhishCheck → Click "Analyze"
3. Review: Attack Score, Authentication, Threat Intel

## Link Analysis
1. Copy suspicious URL (right-click → Copy link)
2. Paste in Link Analysis → Click "Analyze"
3. Review: Screenshot, ML verdict, VirusTotal

## File Analysis
1. Upload file OR enter SHA256/MD5 hash
2. Click "Analyze"
3. Review: VirusTotal detections, Hybrid Analysis sandbox

## Attack Score (0-100)
- 0-20: Safe
- 20-40: Low risk
- 40-70: Medium risk
- 70-100: High risk - Don't interact!
